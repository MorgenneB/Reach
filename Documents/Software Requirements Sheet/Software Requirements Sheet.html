Reach - Software Requirements

1. Introduction

Reach is a web based application that enables its users to locate events occurring within a radius around their current location. The problem it is trying to solve is that, the ability to find events that are happening nearby within a short term is dependent on having a big network of friends in which to reach out to. Often times this network is unreliable in the short term and Reach aims to solve this problem by giving users an alternative to a reliance on their network of friends

1.3 Definitions

Search: An action to curate a list of events that are scheduled to begin or have already begun

Post: An action to introduce an event into the database to be advertised

Host: A user who has scheduled an event

Guest: A user who has not scheduled an event

User Radius: The user defined radius in kilometers around their current location in which to search or advertise
