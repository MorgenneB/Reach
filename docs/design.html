5. Detailed Design

Detailed description of important functions for Reach.

createAccount()

Deposits a User object into the database, constructed using the given username, email, and password parameters, then calls loginAccount().

loginAccount(String, String)

Searches the database for the corresponding account given the username and password. If successful, go to main page. Otherwise, ask to try again.

onMapReady(GoogleMap)

Called when the map is ready to be used. Searches for all events within the user's radius and displays them as Markers on the map.

onMarkerClick(Marker)

Called when a marker is clicked/tapped. Goes to the EventView page.

refresh()

Iterate through the list of events in the database. For each event, if result of distanceBetween() is less than or equal to the user's radius, a Marker will be placed via addMarker().
