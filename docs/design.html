5. Detailed Design

Detailed description of important functions for Reach.

createAccount(String, String, String)

Deposits a User object into the database, constructed using the given username, email, and password parameters, then calls loginAccount().

loginAccount(String, String)

Searches the database for the corresponding account given the username and password. If successful, go to main page. Otherwise, ask to try again.

onMapReady(GoogleMap)

Called when the map is ready to be used. Overriden to call refresh().

onMarkerClick(Marker)

Called when a marker is clicked/tapped. This will be overridden to go to the EventView activity.

refresh()

Iterate through the list of events in the database. For each event, if distanceTo(Location) returns a value (in meters, will have to convert) less than or equal to the user's radius, a Marker will be placed via addMarker(). Location objects can be constructed with an empty String provider as the parameter, and given its latitude/longitude values via its setters.

interested()

Called when the user presses the 'Interested' button. Increments the Event's "totalInterested" value by 1.

giveRating(float)

Called when the user presses the 'Rate' button. Sends the parameter value to the database.

setRadius()

Set's the user's radius to whatever value in kilometers was entered in the text box.
